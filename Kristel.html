HISTORIA DE HTML



HTML en español se traduce como Lenguaje Marcado de Texto, es decir, un lenguaje con marcas que va a permitir estructurar y organizar el texto para que este sea legible. Además, permite, por medio de las etiquetas, establecer ciertas relaciones, hipervínculos y otras modificaciones.

A principios de la década de los 90, el físico Tim Berners Lee, investigador de la Organización Europea para la Investigación Nuclear (CERN), realizó una publicación titulada «HTML Tags», conocida como el origen de HTML. Además, unos años después, fundó la W3C, que es uno de los principales agentes en la historia de HTML, como verás más adelante.

Esta publicación respondía a la necesidad de crear un sistema de hipertexto que les permitiera a los científicos e investigadores del CERN intercambiar información de forma rápida y fácil de referenciar.

VERSIONES DE HTML

HTML 1.0

La versión básica de HTML admite elementos básicos como controles de texto e imágenes. Esta era la versión muy básica de HTML con menos soporte para una amplia gama de elementos HTML.

HTML 2

La versión HTML 2.0 se desarrolló en 1995 con la intención básica de mejorar la versión 1.0 de HTML Ahora se comenzó a desarrollar un estándar para mantener reglas y regulaciones comunes en diferentes navegadores. HTML 2.0 ha mejorado mucho en términos de etiquetas de marcado.

HTML 3.2

Fue desarrollado en 1997. Después de que se desarrolló HTML 2.0, la siguiente versión de HTML fue 3.2 Con la versión 3.2 de HTML, las etiquetas HTML se mejoraron aún más. Vale la pena señalar que debido al mantenimiento estándar del W3C, la versión más nueva de HTML era 3.2 en lugar de 3.

HTML 4.01

Fue desarrollado en 1999. Extendió el soporte de las hojas de estilo en cascada. En la versión 3.2, CSS se incrustó en la página HTML. Por lo tanto, si el sitio web tiene varias páginas web para aplicar al estilo de cada página, debemos colocar CSS en cada página web.

HTML5

Esta es la última versión de HTML. Para un desarrollador, podría usarse en 2014. Se le ocurrió una gran cantidad de etiquetas HTML compatibles.

CONCLUSIONES

Hay muchas versiones de HTML que se están desarrollando. Desde una versión inicial de 1.0 hasta la última versión de 5.2, HTML se ha desarrollado mucho. W3C también ha mantenido estándares para que todos los navegadores puedan tener un estándar común a seguir. HTML5 se ha desarrollado mucho con nuevas etiquetas y soporte de elementos de formulario.
